dywapitchtrack
Dynamic Wavelet Algorithm Pitch Tracking library

Copyright (c) 2010 Antoine Schmitt as@schmittmachine.com

-- Documentation

See the dywapitchtrack.h file for the library API documentation.


-- Version history

Inital release : Antoine Schmitt April 2010
jan 2012 : Minor update : changed the link to the Eric Larson and Ross Maddox paper
apr 2012 : Included the SpeakHere iPhone app example

-- Building, Installing

- the algorithm can be included directly in your project by including the .h and .c files. 
- the iPhone/dywapitchtrack.xcodeproj builds an iPhone static library, against which one can statically link an iPhone application.
